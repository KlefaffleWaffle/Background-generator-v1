# Problem Statement
I have an editorial youtube show featuring me as a talking head. While I often times will use images and video clips to make the video more dynamic, engaging and visually interesting, fans still complain that there's not enough going on on screen, and to some extent, I do agree. But finding and adding good images takes time in an already lengthy editing process. 

# Solution
With this software, I can create a procedurally generated background unique to each video. This makes the video more visually interesting without eating up my time in searching and editing.
