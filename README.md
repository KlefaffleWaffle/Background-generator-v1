# Background Generator
A Python tool that procedurally generates unique animated backgrounds for video content, built with OpenCV, NumPy, and Python's standard math and random libraries.

# Motivation
I produce an editorial YouTube show where visual engagement is important, but sourcing and editing supporting imagery is time-consuming. This tool solves that problem by automatically generating a unique, dynamic background for each video — saving editing time without sacrificing visual quality.
